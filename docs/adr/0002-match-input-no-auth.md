# 対局入力は認証不要にする

対局結果の記録（対局入力）は、ログインなしで誰でも行えるようにする。選手登録・卓登録・対局削除などの他の書き込み操作はオーナー専用だが、対局入力だけは例外とする。

麻雀大会では複数の参加者が手元のデバイスからスコアを記録する運用が一般的であり、全員に Google ログインを強制するのは現実的ではない。大会IDを知っている人だけがアクセスできるという前提が、事実上のアクセス制御として機能している。

## Consequences

- 誤入力のリスクがある。修正（対局削除→再入力）はオーナーだけが行える設計でリスクを限定する。
- 大会IDが漏れると見知らぬ人が対局を入力できる。これは許容リスクと判断した。
