## 小次郎麻雀大会スコア v1.0.0

麻雀大会のスコアをリアルタイムで記録・共有できる Web アプリの初回リリース。

### 実装機能

- 大会の作成・参加（カスタムID対応）
- プレイヤー・卓の登録・リネーム
- 対局結果の入力（スコア入力でウマ・素点を自動計算）
- ランキング自動集計（合計pt・平均pt・平均順位・トップ率など）
- 対局履歴の表示・削除
- 管理者パスワードによる大会削除保護

### 開発方針

想定ユーザーが身内のみで、かつ漏洩しても問題になる情報を扱わないアプリケーションであるため、セキュリティ対策よりも機能実装とデプロイを優先した。
